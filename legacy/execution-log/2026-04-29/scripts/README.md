# Archived X-Run Scripts

These scripts supported closed X-145 to X-175 one-off verdict packets. They are
kept for traceability and replay, but they are not active top-level helper
scripts.

If a future lane needs one of these ideas again, promote the reusable logic into
`src/diffaudit/` or a generic CLI/helper under `scripts/` instead of editing the
archived one-off script in place.

The top-level X-90 replay helpers remain in `scripts/` because they are still
useful for the G1-A / X-90 internal auxiliary evidence boundary.

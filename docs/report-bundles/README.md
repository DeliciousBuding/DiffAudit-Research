# Report Bundles

这里存放需要长期保留的外部模型批量输出结果。

只保留 `raw results / final reports / necessary bundle index`。

不要把临时 `prompt/context/work package`、一次性外发材料、个人整理稿放回这里。

## 当前目录

- `gpt54/`
  - `round1-results/`：GPT-5.4 第一轮原始结果
  - `round2-results/`：GPT-5.4 第二轮原始结果与扩展问题结果

## 使用约定

- 正式规划结论写回 `ROADMAP.md` 和 `docs/comprehensive-progress.md`
- 需要 fresh-session 接手时，优先读取原始结果包，而不是依赖临时综合稿
- 如果后续还有新的外部报告批次，按模型/批次单独建目录，保持结果与工作包分离

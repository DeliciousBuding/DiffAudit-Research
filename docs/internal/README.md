# 内部研究资料

此目录存放维护者需要的资料，不属于公开入门路径。

可能包含：

- 长期进度台账
- Agent 提示词和操作记录
- 比赛时期材料
- 论文阅读报告
- 外部模型审查包
- 规划草稿和退役研究包笔记

不要将这些文件作为产品文案引用。如果某个发现需要对外展示，
请改写后放入 [../evidence/](../evidence/) 或
[../product-bridge/](../product-bridge/)，并注明明确的证据状态。

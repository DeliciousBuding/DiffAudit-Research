## 当前判断

1. 当前仓库不是“只做黑盒”，而是“黑/灰/白三线统一规划、分层执行”；其中灰盒 `PIA` 仍是当前第一优先主讲线，黑盒当前只推进 `recon` 主证据冻结 + `CopyMark` 边界收口 + 频域解释层固化，属于 `writing-only / non-GPU / no admitted change / no new execution line`。

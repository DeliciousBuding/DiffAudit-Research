```powershell
  -C <DIFFAUDIT_ROOT>\Download\datasets-readable\cifar10
```

## GPU Commands For Later

Run these only after GPU use is explicitly allowed.

```powershell

  --ckpt <DOWNLOAD_ROOT>/checkpoints/ddpm-cifar10-750k/checkpoint-step800000.pt `

  --out outputs/h1-scout-800k-same-trajectory-n512 `
```

Stable Diffusion 黑盒成员推理攻击复现材料目录说明

推荐从 README.md 开始查看。

顶层目录说明：

1. artifacts
   最终验收结果，包括：
   - result.csv：5000 张图片逐样本结果，包含 image_path / label / score / prediction
   - metrics.json：最终 AUC / ASR / TPR@FPR=1%
   - roc_curve.png：ROC 曲线图
   - roc_curve.csv：ROC 曲线数据

2. coco_data
   COCO2017-val 非成员样本数据和 annotations。

3. stable_diffusion_data
   LAION-style member 样本数据、member 列表和 BLIP captions。

4. SD_MIA_Reproduction
   核心复现代码，包括 attack.py、components.py、dataset.py、score_single_image.py 等。

5. runs
   精简后的最终运行结果副本，便于查看最终 detector 的验证输出。

6. 论文复现攻击材料_20260516
   攻击阶段材料，包括最终 detector、合并后的 ReDiffuse 特征 NPZ、验证 CSV。

7. 论文复现最终材料_20260516
   汇报/验收用最终材料，包括 README、状态说明、AUC 提升计划、论文 PDF、最终结果表和 ROC 图。

最终指标：

- AUC = 0.7103
- ASR = 0.6846
- TPR@FPR=1% = 0.0716

注意：

本项目不训练 Stable Diffusion，也不微调 LoRA / DreamBooth / Textual Inversion。
使用预训练模型 CompVis/stable-diffusion-v1-4 做 ReDiffuse 风格重构并计算 SSIM 特征。
模型权重未打包在本目录中，需要本机 HuggingFace cache 已有该模型，或联网下载。

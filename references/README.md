# References

This directory stores project-facing reference indexes and mirrored study materials.

## Structure

- `materials/`: mirrored PDF materials copied from the local survey folder
- future notes or reading summaries can also live here

## Usage

Treat these materials as study context for the repository, not as proof that every cited method has already been reproduced.

The current materials index is in [materials/README.md](materials/README.md).
